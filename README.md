# simple_ftp
Creates a Simple FTP, HTTP server.

## Prerequisite
1. Node version 7
2. OS - Linux/MAC
3. Arch - 64 bits

## Installation
1. https://github.com/MVMS1994/simple_ftp/releases

## Usage
Goto the directory that you want to start ftp and http server
#### Linux
`sudo server-linux`

#### MacOS
`sudo server-macos`
